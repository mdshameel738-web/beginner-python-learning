# 🐍 My Python Learning Journey

This repository contains small Python projects I've built while learning to code. I'm a beginner, and this is where I practice what I've learned by actually building things — mostly simple games and calculators using core Python concepts.

## 👋 About Me

I'm learning Python and using these projects to practice things like variables, loops, conditionals, functions, dictionaries, and randomness. Each project below represents something new I picked up along the way.

## 📁 Projects

### 1. Treasure Hunt Game
- **File:** `tresure_hunt_game.py`
- **What it does:** A text-based choose-your-own-adventure game where you make decisions (left/right, swim/wait, pick a door) that lead to winning or losing.
- **Concepts practiced:** `input()`, if / elif / else, string comparison
- **How to run:**
  ```bash
  python tresure_hunt_game.py
  ```

### 2. Band Name Generator
- **File:** `bandgame.py`
- **What it does:** Asks for your hometown and pet's name, then combines them to generate three band name ideas.
- **Concepts practiced:** variables, string concatenation, `input()`
- **How to run:**
  ```bash
  python bandgame.py
  ```

### 3. Bill Dividing Calculator
- **File:** `bill_dividing_calculator.py`
- **What it does:** A tip calculator that works out the tip amount and splits the total bill evenly among a group of people.
- **Concepts practiced:** `float()` / `int()` conversion, arithmetic operations, formatted output
- **How to run:**
  ```bash
  python bill_dividing_calculator.py
  ```

### 4. Rock Paper Scissors
- **File:** `rockpaperscissor_game.py`
- **What it does:** Classic rock-paper-scissors against the computer, complete with ASCII art for each choice.
- **Concepts practiced:** `random.randint()`, if / elif / else, boolean logic
- **How to run:**
  ```bash
  python rockpaperscissor_game.py
  ```

### 5. Password Generator
- **File:** `password_generator.py`
- **What it does:** Generates a random, secure password based on how many letters, numbers, and symbols you want.
- **Concepts practiced:** lists, `random.sample()`, `random.shuffle()`, string joining
- **How to run:**
  ```bash
  python password_generator.py
  ```

### 6. Hangman
- **File:** `hang_man_game.py`
- **What it does:** The classic word-guessing game with a word bank, limited lives, and ASCII art stages that update as you lose lives.
- **Concepts practiced:** `random.choice()`, loops, lists, string building, tracking game state
- **How to run:**
  ```bash
  python hang_man_game.py
  ```

### 7. Silent Auction
- **File:** `silent_auction.py`
- **What it does:** Lets multiple bidders enter their name and bid, then determines and announces the highest bidder.
- **Concepts practiced:** dictionaries, functions, while loops, `input()`
- **How to run:**
  ```bash
  python silent_auction.py
  ```

### 8. Blackjack
- **File:** `blackmancard_game.py`
- **What it does:** A simplified Blackjack game against the computer — deals cards, lets you hit for an extra card, and compares scores to determine the winner.
- **Concepts practiced:** `random.choice()`, lists, conditionals, comparison logic
- **How to run:**
  ```bash
  python blackmancard_game.py
  ```

### 9. Higher or Lower (Celebrity Net Worth)
- **File:** `higherlower_clashgame.py`
- **What it does:** Shows two random celebrities and asks you to guess who has the higher net worth. Keeps score until you get one wrong.
- **Concepts practiced:** dictionaries, `random.choice()`, while loops, score tracking
- **How to run:**
  ```bash
  python higherlower_clashgame.py
  ```
